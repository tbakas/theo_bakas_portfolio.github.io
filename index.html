Vancouver Crime
Exploratory Data Analysis

I'm interested in living and/or working in Vancouver. And I commonly hear about the city's high crime levels. So using data from the Vancouver Police Department, I investigated what neighbourhoods have the least crime. I created a dashboard with Tableau for visualizing the results. I also investigated other questions of potential interest. For example, what types of crime are the most common? And which months tend to have the most crime? Also, I used Python to combine data for different years and transform variables. The code I wrote can be found here.
